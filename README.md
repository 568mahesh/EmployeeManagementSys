Employee Management Full Stack Project

Folders:
1. employee-backend  -> Spring Boot + JPA + MySQL backend
2. employee-frontend -> ReactJS frontend

How to run backend:
1. Create MySQL database: employeedb
2. Update src/main/resources/application.properties with your MySQL password
3. Open terminal inside employee-backend
4. Run: mvn spring-boot:run

How to run frontend:
1. Open terminal inside employee-frontend
2. Run: npm install
3. Run: npm start

Backend URL:
http://localhost:8080/api/employees

Frontend URL:
http://localhost:3000
